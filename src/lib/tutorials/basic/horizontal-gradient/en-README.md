# Horizontal Gradient

Create a simple left-to-right gradient using vUv.x and mix().

## Learning Objectives
- Use vUv.x as a 0-1 factor
- Blend two colors with mix()

## Exercise
- Compute t from vUv.x
- Use mix(a, b, t) and output gl_FragColor

## Hints
- t should be 0 on the left and 1 on the right
